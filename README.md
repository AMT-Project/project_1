## Build image
Run inside projet folder the command:
```bash
  ./build-image.sh
```

If everything goes well, you can run
```bash
docker-compose up
```

Now, your terminal is stuck. Great. Open <http://localhost:9080>
Additionnaly, you can open <http://localhost:9443>
